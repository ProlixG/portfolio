나를 소개합니다.

개발도 잘해야 즐길 수 있다.

     내가 원하는 것을 구현할 수 있을 때 개발자는 흥미와 재미를 느낀다고 생각합니다.
어렸을 적 나만의 게임을 만들고 싶어 스타1 유즈맵을 제작해본 경험이 있습니다. 또한 흥미로운 기능을 구현한 맵이 있다면 언팩하여 어떻게 구현하였는지 찾아보고 사용해보기도 하였습니다. 이렇게 이미 구현된 기능이더라도 필요할 때 활용하고 나만의 것으로 만들기 위해 노력하고있습니다.

     처음 프로그래밍을 시작할 때 아무것도 모르기에 구현할 수 있는 것은 한정되어있습니다. 하나 둘 씩 기능을 배우고 활용 및 응용할 때 원하는 것을 구현할 수 있습니다. 이렇듯이 계속 사용하고 새로운 것을 공부하여 능수능란하게 코딩하며 잘하는 개발자가 되고싶습니다.

교육사항

부산 서울게임아카데미

2018.09.20 ~ 2019.01.14

게임 프로그래밍

울산에서 부산으로 주 5일 일일 2시간 수업을 들으며 C, C++와 win32 API를 배웠습니다.
C, C++로 콘솔 RPG게임을 만들며 프로그래밍을 시작했습니다.
win32 API에서는 흰색 윈도우창을 시작으로 렉트를 이용하여 렉트 충돌, 점프와 중력 기능 구현, 이미지 및 프레임 애니메이션 등을 배웠습니다.
게임 프로그래밍인만큼 예외처리와 한정된 컴퓨터 자원인 메모리 관리가 중요하다는 것을 알게되었습니다.

포트폴리오

웹 페이지

Spring Frame Work를 기반으로 답글형 게시판 웹 페이지를 만들었습니다.
답글 작성 시 최신글이 아래로 내려가도록, 게시글 삭제 시 클릭이 불가능하게 설계했습니다.

  • Bootstrap5를 사용하여 반응형 웹 페이지 구현했습니다.
  • JDBC를 사용하여 오라클 데이터베이스와 연동했습니다.
  • Ajax를 사용하여 페이지 새로고침 없이 댓글을 구현했습니다.
  • JUnit5와 Postman을 통해 단위테스트 및 Rest API테스트를 했습니다.
View Project

[팀프로젝트] 꽃미힐미

4명이서 Spring Frame Work를 기반으로 제작한 정기구독 신청사이트입니다.
팀 작업 내에 백엔드 개발 및 버그픽스, 영상녹화 그리고 백엔드부분 PPT 발표를 맡았습니다.

  • 오픈API로 팝업 도로명 주소 API를 구현했습니다.
  • MyBatis를 사용하여 SQL 쿼리 및 객체 맵핑을 관리했습니다.
  • Rest API를 사용하여 URL 매핑과 Http Method를 관리했습니다.
  • sun.javaMail API를 사용하여 이메일 서비스를 구현했습니다.
  • JUnit5와 Postman을 통해 단위테스트 및 Rest API테스트를 했습니다.
